## Organisation Roles & Permissions

You build your organisation the way you want. Create teams, invite or remove members, assign clear roles, and keep tabs on important changes with activity logs - so everyone knows who’s responsible for what.

<img src="/images/screenshots/organisations-roles.png" alt="Organisations Roles & Permissions" />

## Player How‑To

- Create or join (via invitation) an organisation
- Assign roles to members and set permissions
- Create teams
- Review activity logs for important changes

## Admin Capabilities

- Configure org role templates and permission sets
- Moderate membership changes and ownership transfers
- Audit sensitive actions via activity logs
